# Pruebas de Usabilidad y Accesibilidad

## Usabilidad
Se evaluó:
- claridad de navegación
- facilidad de localización de funciones
- consistencia visual
- comprensión de mensajes
- facilidad de edición y cancelación

## Accesibilidad básica
Se revisó:
- contraste
- legibilidad
- tamaño de botones
- orden visual
- estructura coherente de formularios

## Resultado
El sistema cumple adecuadamente para un prototipo de alta fidelidad académico.

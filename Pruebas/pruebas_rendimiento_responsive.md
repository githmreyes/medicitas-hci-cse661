# Pruebas de Carga, Rendimiento y Diseño Responsivo

## Rendimiento
En entorno local, el sistema responde adecuadamente en:
- carga de login
- carga de dashboard
- listados CRUD
- reportes básicos

## Responsive
Se evaluó:
- sidebar colapsable
- adaptación de tablas
- redistribución de formularios
- visualización en tamaños medianos y pequeños

## Resultado
Comportamiento satisfactorio para fines de demostración y validación académica.

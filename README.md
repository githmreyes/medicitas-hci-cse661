# MediCitas HCI

Sistema de Gestión de Citas de Especialidades Médicas centrado en HCI.

## Tecnologías
- PHP
- MySQL
- phpMyAdmin
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Login y logout
- Dashboard
- CRUD de pacientes
- CRUD de médicos
- CRUD de citas
- Cancelación de citas
- Historial de citas
- Reportes básicos
- Mensajes flash
- Validaciones

## Instalación local
1. Copiar la carpeta `MEDICITAS_HCI` en `htdocs`.
2. Importar `database_medicitas_hci.sql` en phpMyAdmin.
3. Verificar datos en `config/database.php`.
4. Abrir:
   `http://localhost/MEDICITAS_HCI/views/auth/login.php`

# Pruebas Preliminares de Usabilidad y Accesibilidad

## Usabilidad
Se revisaron:
- claridad del menú lateral
- comprensión de formularios
- secuencia de tareas
- visibilidad de acciones
- mensajes de retroalimentación

## Accesibilidad básica
Se revisaron:
- contraste
- tamaño de controles
- textos legibles
- estructura visual ordenada
- consistencia de componentes

## Hallazgo general
El sistema presenta una interfaz comprensible y usable para operaciones básicas del caso de estudio.

# Guía de Uso del Prototipo

## Ingreso al sistema
1. Abrir el login
2. Ingresar correo y contraseña
3. Acceder al dashboard

## Flujo sugerido de prueba
1. Registrar paciente
2. Registrar médico
3. Crear cita
4. Editar cita
5. Cancelar cita
6. Revisar historial
7. Consultar reportes

## Mensajes del sistema
- éxito: operación completada
- danger: error o credenciales inválidas
- warning: cancelación o eliminación

## Recomendación
Usar primero datos demo en entorno local para poblar el sistema antes de evaluar reportes.

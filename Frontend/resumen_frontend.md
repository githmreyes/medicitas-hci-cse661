# Resumen Frontend

La interfaz del sistema fue desarrollada con HTML5, CSS3, Bootstrap 5, Bootstrap Icons, JavaScript y PHP embebido. Se diseñó una experiencia cercana a un sistema real mediante sidebar, topbar, dashboard, tablas modernas y formularios con jerarquía visual clara.

## Objetivos frontend
- facilitar la navegación
- mejorar la comprensión de tareas
- reducir errores de uso
- ofrecer una experiencia clara y consistente

## Pantallas implementadas
- login
- dashboard
- listado de pacientes
- nuevo paciente
- editar paciente
- listado de médicos
- nuevo médico
- editar médico
- listado de citas
- nueva cita
- editar cita
- historial de cita
- reportes

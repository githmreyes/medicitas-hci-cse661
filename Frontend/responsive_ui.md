# Página Responsive

## Enfoque
La interfaz fue construida con Bootstrap 5 y estilos CSS personalizados para adaptarse a escritorio, tableta y móvil.

## Elementos responsive
- sidebar colapsable
- tablas con scroll horizontal
- formularios adaptables por columnas
- topbar simplificada en móvil

## Beneficio
Permite que el sistema siga siendo utilizable en distintos tamaños de pantalla, lo que mejora accesibilidad y experiencia de usuario.

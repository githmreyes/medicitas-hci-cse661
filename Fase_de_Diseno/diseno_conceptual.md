# Diseño Conceptual

## Nombre del sistema
MediCitas HCI

## Descripción general
MediCitas HCI es un sistema web de gestión de citas de especialidades médicas, diseñado bajo principios de Interacción Humano-Computadora (HCI), con énfasis en usabilidad, accesibilidad, claridad visual y trazabilidad operativa.

## Problema que resuelve
En muchos entornos médicos, la programación y seguimiento de citas se realiza de manera manual o en sistemas poco intuitivos, generando duplicidad, errores de agenda, dificultad de seguimiento y una experiencia deficiente para pacientes y personal administrativo.

## Objetivo funcional
Permitir el registro de pacientes, gestión de médicos, programación de citas, edición, cancelación, historial y reportes básicos dentro de una interfaz clara y consistente.

## Usuarios del sistema
1. Administrador
2. Recepcionista
3. Médico especialista
4. Paciente

## Principios HCI aplicados
- Visibilidad del estado del sistema
- Prevención de errores
- Consistencia y estándares
- Reconocimiento antes que recuerdo
- Navegación clara
- Retroalimentación inmediata
- Reducción de carga cognitiva

## Módulos principales
- Login y autenticación
- Dashboard
- Gestión de pacientes
- Gestión de médicos
- Gestión de citas
- Historial de citas
- Reportes

## Requerimientos funcionales principales
- Registrar pacientes
- Registrar médicos
- Gestionar citas
- Editar y cancelar citas
- Consultar historial
- Mostrar reportes básicos
- Validar accesos por sesión

## Requerimientos no funcionales
- Interfaz responsive
- Seguridad básica con autenticación
- Validación en cliente y servidor
- Diseño consistente
- Base de datos relacional
- Facilidad de uso

## Enfoque de diseño
Se adoptó una estructura visual basada en sidebar, topbar, tarjetas de indicadores, tablas modernas y formularios jerarquizados. El objetivo fue acercar el prototipo a un entorno real de operación clínica y administrativa.

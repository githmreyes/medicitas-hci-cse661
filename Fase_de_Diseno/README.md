# Fase de Diseño – MediCitas HCI

Esta carpeta contiene la documentación de la fase de diseño del prototipo de alta fidelidad del caso de estudio **Sistema de Gestión de Citas de Especialidades Médicas centrado en HCI**.

## Contenido
- Diseño conceptual
- Modelo relacional de la base de datos
- Diagrama de casos de uso
- Arquitectura de software
- Mapa de navegación

## Objetivo
Definir de manera visual, lógica y funcional la estructura del sistema antes de su implementación, aplicando principios de usabilidad, accesibilidad y diseño centrado en el usuario.

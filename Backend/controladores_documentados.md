# Controladores Documentados

## authController.php
Responsable de:
- autenticar usuario
- crear sesión
- cerrar sesión
- validar credenciales

## pacienteController.php
Responsable de:
- guardar paciente
- actualizar paciente
- eliminar paciente
- validar campos requeridos

## medicoController.php
Responsable de:
- guardar médico
- actualizar médico
- eliminar médico
- validar datos mínimos

## citaController.php
Responsable de:
- guardar cita
- actualizar cita
- cancelar cita
- registrar historial
- validar fecha, hora, paciente y médico

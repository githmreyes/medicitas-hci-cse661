# Resumen Backend

El backend del sistema fue desarrollado en PHP bajo una estructura modular que separa configuración, modelos, controladores y vistas. La persistencia se implementó mediante MySQL y conexión PDO.

## Tecnologías
- PHP 8+
- MySQL
- PDO
- phpMyAdmin

## Funcionalidades implementadas
- Login y logout
- Protección de sesión
- CRUD de pacientes
- CRUD de médicos
- CRUD de citas
- Historial de citas
- Mensajes flash
- Validaciones de servidor

## Objetivo del backend
Garantizar consistencia de datos, seguridad básica, manejo estructurado de operaciones y soporte directo a la interfaz de usuario.
